# Iris Classification Machine (4 Species)

Automated machine learning classifier for identifying four types of Iris flowers. This project was developed as part of the AI & ML Course, Assignment 1.

## Project Structure
- `iris_v2.csv`: The expanded dataset containing 4 species.
- `classifier.py`: The Python script for training and testing the machine.
- `confusion_matrix_v2.png`: Visual result of the test phase.
- `PRD.md`, `PLAN.md`, `REPORT.md`: Project documentation.

## Requirements
To run this project, you need:
- Python 3.12+
- Pandas
- Scikit-learn
- Matplotlib
- Seaborn

## How to Run
1. Open your terminal in the project folder.
2. Run the classifier:
   ```bash
   python classifier.py
   ```
3. View the results in the console and open `confusion_matrix_v2.png` to see the performance.
